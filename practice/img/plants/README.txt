รูปพืชสำหรับเกม "จับคู่ภาพ–คำ (พืช)" — วิทยาศาสตร์ EP บทที่ 1 Plants
=====================================================================

วางไฟล์รูป 9 ไฟล์นี้ในโฟลเดอร์นี้ (practice/img/plants/)
เกมจะดึงมาแสดงแทนภาพ SVG อัตโนมัติ ถ้าไฟล์ไหนยังไม่มี จะใช้ SVG สำรองให้เอง

ชื่อไฟล์ต้องตรงเป๊ะ (ตัวพิมพ์เล็ก .png):
  banana.png      (กล้วย / Banana plant)
  mango.png       (มะม่วง / Mango tree)
  cactus.png      (กระบองเพชร / Cactus plant)
  coconut.png     (มะพร้าว / Coconut tree)
  sunflower.png   (ทานตะวัน / Sunflower plant)
  hibiscus.png    (ชบา / Hibiscus plant)
  pineapple.png   (สับปะรด / Pineapple plant)
  rose.png        (กุหลาบ / Rose plant)
  papaya.png      (มะละกอ / Papaya tree)

คำแนะนำรูป:
  - พื้นหลังโปร่งใส (transparent PNG) จะสวยที่สุด
  - จัตุรัส (เช่น 512x512) วางต้นไม้ตรงกลาง
  - สไตล์การ์ตูนแบน (flat) สีสด ๆ ให้เหมือนในหนังสือ

------------------------------------------------------------
Prompt สำหรับ Nano Banana / Gemini (คัดลอกไปวางได้เลย)
วางรูปตัวอย่างจากหนังสือแนบไปด้วย แล้วสั่งทีละต้น หรือสั่งเป็นชุด:
------------------------------------------------------------
"Flat cartoon illustration of a single [PLANT NAME] for a kids'
science workbook, in the same clean colorful style as the reference
image. Centered on a fully transparent background, no text, no shadow
box, no border. Square composition. Bright friendly colors."

แทน [PLANT NAME] ด้วย: banana plant, mango tree, cactus plant,
coconut tree, sunflower plant, hibiscus plant, pineapple plant,
rose plant, papaya tree

* ให้ทั้ง 9 ต้นใช้สไตล์/แสง/เส้นเหมือนกัน จะดูเป็นชุดเดียวกันสวยงาม
* เซฟเป็น PNG พื้นหลังโปร่งใส ตั้งชื่อตามด้านบน แล้ววางในโฟลเดอร์นี้
